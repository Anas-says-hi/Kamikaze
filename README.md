## Controls

+ <kbd>left</kbd> or <kbd>right</kbd> arrow keys or <kbd>a</kbd>/<kbd>d</kbd> keys to move left or right.
+ space or up arrow to shoot.
+ if a ship appears press <kbd>x</kbd> to crash into the ship.
